# Updating versioning for package managers and whatnot

- App metainfo: `extra/linux/dev.fide.fide.metainfo.xml`
- macOS plist (`CFBundleShortVersionString`): `extra/macos/FIDE.app/Contents/Info.plist`
- Rust: `Cargo.toml`
- Obviously changelog: `CHANGELOG.md`
- RPM spec: `fide.spec`
- Windows wix spec (`<Product [...] Version=X.X.X>`): `extra/windows/wix/fide.wxs`
